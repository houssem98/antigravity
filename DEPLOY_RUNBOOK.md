# Deploy Runbook — Antigravity to Production Cloud

**Target:** Ship to cloud with free tier + multi-provider billing (Paddle, PayPal, Payoneer, Crypto). Works from Tunisia.

**Stack:** Fly.io (gravity-api + market-server) · Vercel (market-ui) · Qdrant Cloud (vectors) · Elastic Cloud (BM25) · Neo4j Aura free (graph)

**Cost (monthly):**
- Fly.io (2 apps, shared CPU, 0-1 min machines): ~$5–15
- Qdrant Cloud free tier (1 GB)                  : $0
- Elastic Cloud trial → paid plan               : ~$95
- Neo4j Aura Free                               : $0
- Vercel Hobby                                  : $0
- Domain (one-time)                             : ~$12
- Voyage embed (one-time for indexing)          : ~$4
- **Total**: **~$100–110/mo + $16 one-time**

**Billing:** Free tier (10 searches/day) → Pro ($49/mo, Paddle/PayPal/Payoneer/Crypto) → Team ($499/mo, Paddle/PayPal/Payoneer/Crypto)

---

## 0 · Prep (20 min)

1. **Buy domain** (Namecheap / Porkbun). Example: `antigravity.fyi`.

2. **Fly.io account:** https://fly.io → sign up (free tier available).
   ```bash
   # Install CLI
   brew install flyctl              # macOS
   # Or Windows: iwr https://fly.io/install.ps1 -useb | iex
   fly auth login
   ```

3. **GitHub repo:** push current code to main. (Fly auto-deploys on git push if you enable it, or manual `fly deploy`.)

4. **Payment setup (choose providers):**
   - **Paddle** (Merchant of Record, works Tunisia): https://paddle.com → create account
   - **PayPal** (optional): https://developer.paypal.com → create app, get `PAYPAL_CLIENT_ID` + `PAYPAL_CLIENT_SECRET`
   - **Payoneer** (optional): https://payoneer.com → email address for receiving payments
   - **Crypto** (optional): Set BTC/ETH/USDT wallet addresses in admin panel after deploy

5. **Env template for prod:**
   ```
   # Secrets (auto-generated by Fly if not set)
   AUTH_JWT_SECRET=<auto>
   KEY_ENCRYPTION_KEY_V1=<auto>
   SAML_STATE_SECRET=<auto>
   
   # Billing (set manually or leave blank to skip)
   PADDLE_VENDOR_ID=<your vendor ID>
   PAYPAL_CLIENT_ID=<your client ID>
   PAYONEER_EMAIL=<your email>
   ```

## 1 · Provision external services (40 min)

### Qdrant Cloud (vector database)
1. https://cloud.qdrant.io → free 1GB cluster, region nearest you.
2. Copy URL + API key → save as `QDRANT_URL` + `QDRANT_API_KEY`.

### Elastic Cloud (BM25 search)
1. https://cloud.elastic.co → 14-day trial → "Optimized for Search".
2. Region match Qdrant.
3. Save Cloud ID + auth → `ELASTICSEARCH_URL` + `ELASTICSEARCH_USER` + `ELASTICSEARCH_PASS`.

### Neo4j Aura Free (knowledge graph)
1. https://console.neo4j.io → free instance.
2. Save `NEO4J_URI` + `NEO4J_PASSWORD` (user = `neo4j`).

### LLM API Keys
Collect from your `.env`:
- `ANTHROPIC_API_KEY`
- `COHERE_API_KEY`
- `VOYAGE_API_KEY`
- `DEEPSEEK_API_KEY`
- `GROQ_API_KEY`
- (others optional)

## 2 · Deploy gravity-api on Fly.io (30 min)

```bash
cd services/gravity-api

# Launch app (auto-generates fly.toml if needed)
fly launch --name gravity-api-prod --copy-config --no-deploy

# Set secrets (Fly keeps them encrypted)
fly secrets set \
  ANTHROPIC_API_KEY=sk-ant-... \
  COHERE_API_KEY=... \
  VOYAGE_API_KEY=pa-... \
  DEEPSEEK_API_KEY=... \
  GROQ_API_KEY=... \
  QDRANT_URL=https://... \
  ELASTICSEARCH_URL=https://... \
  NEO4J_URI=bolt://... \
  NEO4J_PASSWORD=... \
  PADDL_VENDOR_ID=12345 \
  PAYPAL_CLIENT_ID=... \
  PAYONEER_EMAIL=your@email.com

# Adjust fly.toml if needed:
# - min_machines_running = 1 (always on) or 0 (cold-start, cheaper)
# - memory_mb = 1024 (fine) or 2048 (if loading transformers)

fly deploy
```

Fly auto-provisions Postgres + Redis (included in pricing).

**Health check:**
```bash
curl https://gravity-api-prod.fly.dev/health
# Expected: {"status":"ok"}

fly logs -a gravity-api-prod  # Stream logs
```

**Save the URL** → becomes `VITE_GRAVITY_API_URL` for market-ui.

## 3 · Deploy market-server on Fly.io (20 min)

```bash
cd services/market-server

fly launch --name market-server-prod --no-deploy

# Env vars (same set as gravity-api, + LLM keys for server-side routing)
fly secrets set \
  GRAVITY_API_URL=https://gravity-api-prod.fly.dev \
  ANTHROPIC_API_KEY=sk-ant-... \
  DEEPSEEK_API_KEY=... \
  GROQ_API_KEY=... \
  TAVILY_API_KEY=... \
  ALPHA_VANTAGE_API_KEY=...

fly deploy
```

**Health check:**
```bash
curl https://market-server-prod.fly.dev/api/health
```

**Save the URL** → becomes `VITE_API_URL` for market-ui.

---

## 4 · Deploy market-ui on Vercel (15 min)

1. https://vercel.com → import repo. Set root dir = `apps/market-ui`.
2. Framework = Vite (auto-detected).
3. Env vars:
   ```
   VITE_GRAVITY_API_URL=https://gravity-api-prod.fly.dev
   VITE_API_URL=https://market-server-prod.fly.dev
   VITE_AUTH_BACKEND=gravity_api
   VITE_DEV_AUTH_BYPASS=false
   VITE_TAVILY_API_KEY=<optional for direct frontend calls>
   VITE_ALPHA_VANTAGE_API_KEY=<your key>
   ```
4. Deploy. Vercel gives `<slug>.vercel.app`.
5. Add custom domain: Vercel → Settings → Domains → `antigravity.fyi`. Point DNS per Vercel.
6. Site live at `https://antigravity.fyi`.

## 6 · Multi-provider billing setup (30 min)

**Free tier is auto-enabled.** New users get 10 searches/day limit without paying.

Verify free tier:
```bash
# Sign up at https://antigravity.fyi/auth
# Check /billing page — should show "Free" plan
# Try 11 searches in one day — 11th fails with 429 rate limit

# Rate limits enforced in app/api/middleware/rate_limit.py:
# - Per-minute: 10 req/min
# - Monthly: 100 queries/month
```

### Configure payment providers (each optional)

#### Paddle (recommended — works Tunisia, Merchant of Record)
1. https://paddle.com → create account, set up business profile.
2. Get Vendor ID + API key.
3. Fly secret:
   ```bash
   fly secrets set -a gravity-api-prod PADDLE_VENDOR_ID=12345
   ```
4. Checkout creates Paddle link automatically (no webhook needed).

#### PayPal
1. https://developer.paypal.com → create app.
2. Get `Client ID` + `Secret`.
3. Fly secret:
   ```bash
   fly secrets set -a gravity-api-prod \
     PAYPAL_CLIENT_ID=... \
     PAYPAL_CLIENT_SECRET=...
   ```

#### Payoneer (manual, no API)
1. Create Payoneer account, get email.
2. Fly secret:
   ```bash
   fly secrets set -a gravity-api-prod PAYONEER_EMAIL=you@payoneer.com
   ```

#### Crypto (manual wallet deposit)
1. Get BTC/ETH/USDT wallet addresses.
2. After deploy, log in as admin at `/admin/billing`.
3. Go to "Wallets" → edit BTC/ETH/USDT_ERC20/USDT_TRC20 addresses.
4. QR code auto-generated for each wallet.

### Test checkout
```bash
# Log in at https://antigravity.fyi/auth
# Go to /billing → click upgrade → select provider
# Paddle: opens Paddle checkout (test card: 4242)
# PayPal: OAuth redirect
# Payoneer: manual transfer instructions
# Crypto: shows QR code + wallet address
```

## 7 · Smoke test (30 min)

From a clean browser:
1. Visit `https://antigravity.fyi/auth`.
2. Sign up with email + password.
3. Confirm you're on free plan (10 searches/day).
4. Submit query "What was Apple revenue in FY 2024?" — verify answer + citations.
5. Try 11 searches today — 11th should fail with 429 (rate limit).
6. Click `/billing` → "Upgrade" → select Paddle (or PayPal/Payoneer/Crypto).
7. Test checkout flow (Paddle: use test card `4242 4242 4242 4242`).
8. Confirm subscription appears in admin panel at `/admin/billing`.

---

## 8 · Admin panel setup (5 min)

1. Create first user as admin (manual SQL or Supabase console):
   ```sql
   UPDATE auth.users SET metadata = jsonb_set(metadata, '{role}', '"admin"')
   WHERE email = 'your@email.com';
   ```
2. Log in → navigate to `/admin/billing`.
3. Review + customize:
   - **Plans:** change prices, features, search limits, active toggle
   - **Providers:** enable/disable Paddle, PayPal, Payoneer, Crypto
   - **Wallets:** set BTC/ETH/USDT addresses
   - **Subscriptions:** view all users + manually set plan/status
   - **Invoices:** confirm pending crypto invoices
